## Summary
버전 업데이트 시 11개 파일 동시 수정 필수 — 부분 업데이트 금지

## Context
domangcha/VERSION만 수정하거나 일부 CLAUDE.md만 변경하면 버전 불일치 발생.
GATE 3에서 자동 차단되지만, 수정해야 할 파일이 많아 반복적으로 놓치는 패턴.

## Rule
버전 업데이트 시 아래 11개 파일 전부 동시 수정:
1. domangcha/VERSION
2. package.json version
3. domangcha/CLAUDE.md 헤더
4. CLAUDE.md (루트) 헤더 + 버전 라인
5. ~/.claude/CLAUDE.md 헤더
6. domangcha/skills/ceo-system/SKILL.md 헤더
7. domangcha/skills/ceo-core/SKILL.md 헤더
8. domangcha/skills/ceo-sprint/SKILL.md 버전 마커
9. domangcha/skills/ceo-standards/SKILL.md 초기화 메시지
10. README.md 버전 배지
11. git commit: v{VERSION}: 내용
관련: ERR-003

## Example
```
# 위반
domangcha/VERSION: 2.0.45
package.json: still "2.0.44"  ← GATE 3 FAIL

# 처방
모든 파일 동시 sed 또는 Edit 도구로 일괄 수정
```
